A9G-Based SOS Button GPS Tracker

Developed a cost-effective, low-power emergency alert device using ESP32 and A9G GSM/GPRS/GPS module. The system sends real-time GPS location to the cloud via MQTT when the SOS button is pressed, enabling rapid response in critical situations. Integrated with Adafruit IO for cloud management and designed with FreeRTOS for efficient multitasking. Emphasized privacy, manufacturability, and sustainability for widespread, ethical use in personal safety applications. .
